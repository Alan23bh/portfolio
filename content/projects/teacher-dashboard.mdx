## Problem

Teachers needed a quick way to see student performance and class trends.

## Solution

Built a dashboard with KPIs + charts + student management.

## What I built

- KPI cards
- Charts (ApexCharts)
- CRUD students
- E2E tests with Playwright + CI

## Lessons learned

- Stable locators for E2E
- Chart components + state management patterns
  <Callout title="Key takeaway">
    Playwright locators + a small Page Object pattern made the E2E tests stable
    and readable.
  </Callout>
